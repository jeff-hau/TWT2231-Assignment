Welcome to the Air Quality Website

**Note:You cannot directly load the webpage from index.html

To run the webpage:

  • Ensure you have Node.js installed.
  • Open the folder use VS Code.
  • Open the terminal in VS Code (Can use Ctrl + `)
  • Run npm install to install dependencies.
  • Use npm start to start the development server.
  • Open http://localhost:3000 in your browser to view the app.
  • Alternatively, you can visit the page using this URL for the Webpage: GitHub TWT2231 Assignment
